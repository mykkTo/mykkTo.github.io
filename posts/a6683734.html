Doris操作指南

数据库 docker doris

一、前言

站长最近入职了一家做数据中台的项目，数据库用到的是Doris ,于是接手预研，

目前主要写一些基础用法，后续如果项目又扩展，也还在做这块及相关，文章会有进阶和丰富的地方。

一、搭建

1、概要

配置一个FE+三个BE的集群，使用版本1.1.5

  1. 下载FE和BE包
  2. 准备FE和BE环境
  3. 配置FE
  4. 配置BE
  5. 在FE中添加BE

以下是一台主机在docker上部署多个容器（fe be ）

2、步骤

1-1、下载FE和BE包

从官方下载已经编译好的包： doris下载

注意：这边如果CPU是 inter E5系列的：要选择 no avx2

  • 配置FE和BE的目录结构；

1-3、配置FE

1、启动FE Docker

运行命令（换行是为了美观，自己拷贝请合为一行）：

docker run -it -d --name=doris-fe -p 18030:8030 -p 19030:9030 -v /root/docker/doris/fe/apache-doris-fe-1.1.5-bin:/opt/doris/fe -v /root/docker/doris/fe/apache-doris-fe-1.1.5-bin/doris-meta:/opt/doris/doris-meta apache/doris:build-env-for-1.1.0
host> docker run -it -d --name=doris-fe
-p 18030:8030
-p 19030:9030
-v /root/docker/doris/fe/apache-doris-fe-1.1.5-bin:/opt/doris/fe
-v /root/docker/doris/fe/apache-doris-fe-1.1.5-bin/doris-meta:/opt/doris/doris-meta
apache/doris:build-env-for-1.1.0

2、进入FE进行配置

host> docker exec -it doris-fe /bin/bash
# 查看FE的ip地址
fe> cat /etc/hosts
# 配置FE的网络
fe> vi /opt/doris/fe/conf/fe.conf
# 取消下面行的注释并把这里的ip网段配置成上面查看的ip地址网段
vi> priority_networks = 172.17.0.0/16

其中 cat /etc/hosts 看到的IP 用于后面再 FE mysql 中配置BE

当下的 priority_networks 配置成网段的 /16 不要丢掉

3、配置FE的JDK：

容器内部的已经有了JDK11 ，所以无需下载，直接配置

以下是3条命令直接合并

fe> alternatives --set java java-11-openjdk.x86_64 && alternatives --set javac java-11-openjdk.x86_64 && export JAVA_HOME=/usr/lib/jvm/java-11

4、配置FE的环境变量：

fe> vi /etc/profile.d/doris.sh
export DORIS_HOME=/opt/doris/fe/
export PATH=$PATH:$DORIS_HOME/bin

# 保存后退出vi
fe> source /etc/profile.d/doris.sh

5、启动FE

fe> start_fe.sh --daemon

6、验证FE

访问FE网站 http://localhost:18030 ，登录账号admin，密码为空。

1-4、配置BE

1、启动BE Docker

docker run --privileged -it -d --name=doris-be-01 -p 9061:9060 -v /root/docker/doris/be-01/apache-doris-be-1.1.5-bin-x86_64:/opt/doris/be apache/doris:build-env-for-1.1.0
host> docker run --privileged -it -d --name=doris-be-01 
    -p 9061:9060 
    -v /root/docker/doris/be-01/apache-doris-be-1.1.5-bin-x86_64:/opt/doris/be 
    apache/doris:build-env-for-1.1.0

–privileged 这里添加了特权模式，跟配置/etc/sysctl.conf的vm.max_map_count=2000000相关。

–name=doris-be-01 多个be对应的名字不同：doris-be-02、doris-be-03

-p 9061:9060 多个be对应的端口不同，9062:9060、9063:9060

2、进入BE进行网络配置

host> docker exec -it doris-be-01 /bin/bash
# 查看FE的ip地址
be> cat /etc/hosts
# 配置BE的网络
be> vi /opt/doris/be/conf/be.conf
# 取消下面行的注释并把这里的ip网段配置成上面查看的ip地址网段
vi> priority_networks = 172.17.0.0/16

3、配置BE的JDK：

be> alternatives --set java java-11-openjdk.x86_64
be> alternatives --set javac java-11-openjdk.x86_64
be> export JAVA_HOME=/usr/lib/jvm/java-11
alternatives --set java java-11-openjdk.x86_64 && alternatives --set javac java-11-openjdk.x86_64 && export JAVA_HOME=/usr/lib/jvm/java-11

4、配置BE的环境变量：

be> vi /etc/profile.d/doris.sh
export DORIS_HOME=/opt/doris/be/
export PATH=$PATH:$DORIS_HOME/bin

# 保存后退出vi
be> source /etc/profile.d/doris.sh

5、配置vm.max_map_count

be> vi /etc/sysctl.conf
vi> vm.max_map_count=2000000
# 保存退出vi
# 生效
be> sysctl -p
# 查看结果
be> sysctl -a | grep vm.max_map_count

6、启动BE

be> start_be.sh --daemon

1-5、同理继续 1-4 步骤，配置和运行其余BE：

运行doris-be-02：

host> docker run --privileged -it -d --name=doris-be-02
    -p 9062:9060 
    -v /root/docker/doris/be-02/apache-doris-be-1.1.5-bin-x86_64:/opt/doris/be 
    apache/doris:build-env-for-1.1.0

运行doris-be-03：

host> docker run --privileged -it -d --name=doris-be-03
    -p 9063:9060 
    -v /root/docker/doris/be-03/apache-doris-be-1.1.5-bin-x86_64:/opt/doris/be 
    apache/doris:build-env-for-1.1.0

1-6、在FE中添加BE

1、在FE环境中安装 mysql客户端：

fe> yum install mysql

2、然后进入通过mysql客户端进入doris配置backends：

fe> mysql -h 127.0.0.1 -P9030 -uroot
# 9050为be的心跳端口heartbeat_service_port
# 扩容
mysql> ALTER SYSTEM ADD BACKEND "172.17.0.3:9050";
mysql> ALTER SYSTEM ADD BACKEND "172.17.0.4:9050";
mysql> ALTER SYSTEM ADD BACKEND "172.17.0.5:9050";

# 缩容
mysql> ALTER SYSTEM DECOMMISSION BACKEND "172.17.0.4:9050";
 
# 查询节点状态 Alive属性true即为正常
mysql> show PROC '/backends';
