前言

1、闲聊

最近博主在准备转大数据，于是学习Flink,去年22年5月有文章，不过更新到 DataStream Api 用法，时间语义和窗口等等还没有更新，

目前又爱上了 Scala ，众所周知Flink 或者是 Spark 这些流批计算框架的翘楚都有它的身影，即使不在大数据，在java 项目开发也可以使用；

scala 和 java ，就像 C and C++ ,其实 scala 也可以称为 java ++ ，它的语法简单，开发高效，其实即使你是java开发也可以玩他

今天博主开了这个专栏，就是想搞了混合开发，下午搞了半天终于搞定 SpringBoot (scala + java) 的jar 打包部署 maven 项目

当然了，这个专栏也会是完整的 scala 学习教程！

pom 文件

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>org.example</groupId>
    <artifactId>scalaTest01</artifactId>
    <version>1.0-SNAPSHOT</version>

    <properties>
        <scala.version>2.12.11</scala.version>
<!--        <spark.version>3.0.1</spark.version>-->
    </properties>

    <dependencies>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
            <version>2.2.7.RELEASE</version>
        </dependency>

        <!--        因为我们需要使用Scala代码，所以我们还需要加入scala-library依赖-->
        <dependency>
            <groupId>org.scala-lang</groupId>
            <artifactId>scala-library</artifactId>
            <version>2.12.11</version>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
                <configuration>
                    <!-- main 方法入口，可以是java 也可以是 scala ,Springboot 项目的话则是主启动器 -->
                    <mainClass>com.JavaDemo</mainClass>
                    <includeSystemScope>true</includeSystemScope>
                </configuration>
                <executions>
                    <execution>
                        <goals>
                            <goal>repackage</goal>
                        </goals>
                    </execution>
                </executions>
            </plugin>

            <plugin>
                <groupId>org.scala-tools</groupId>
                <artifactId>maven-scala-plugin</artifactId>
                <version>2.15.2</version>
                <executions>
                    <execution>
                        <id>scala-compile-first</id>
                        <goals>
                            <goal>compile</goal>
                        </goals>
                        <configuration>
                            <includes>
                                <include>**/*.scala</include>
                            </includes>
                        </configuration>
                    </execution>
                    <execution>
                        <goals>
                            <goal>compile</goal>
<!--                            <goal>testCompile</goal>-->
                        </goals>
                        <configuration>
                            <jvmArgs>
                                <!-- 需要的配置 -->
                                <jvmArg>-Xss8m</jvmArg>
                            </jvmArgs>
                        </configuration>
                    </execution>
                    <execution>
                        <id>scala-test-compile</id>
                        <goals>
                            <goal>testCompile</goal>
                        </goals>
                    </execution>
                </executions>
            </plugin>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>2.3.2</version>
                <configuration>
                    <source>1.8</source>
                    <target>1.8</target>
                    <encoding>UTF-8</encoding>
                </configuration>
            </plugin>
        </plugins>
    </build>

</project>

注意：scala 和 java 要分开，不然编译会出问题

1、JavaDemo.java ，java代码调用 scala

package com;
import com.kk.chapter01.Car;

public class JavaDemo {
    public static void main(String[] args) {
        Car car = new Car (2009);

        System.out.println(car);
        car.drive(10);
        System.out.println("java:"+car);
    }
}

2、Car.scala (Object)

package com.kk.chapter01

class Car(val year: Int) {
  private[this] var miles: Int = 0

  def drive(distance: Int) {
    miles += distance
    println("scala:" + miles)
  }

  override def toString(): String = "year:" + year + " miles:" + miles
}

3、TestScala$.scala (class)

package com.kk.chapter01

object TestScala {
  def main(args: Array[String]): Unit = {

    var str1: String = new String("hello")
    var str2: String = "hello"
    println(str1 == str2)

    var age: Int = 25
    var sal: Float = 3.2f
    var height: Double = 176.51

    // 格式化字符串
    printf("名字--%s 年龄%d 薪资%f 身高%f", str1, age, sal, height)
    println()

    // scala 支持使用 $输出内容，编译器会解析对应 $变量
    println(s"当下的信息：\n$str1,\t年龄：$age\t薪资：$sal\t身高：$height")
    // 添加计算内容
    println(s"当下的信息：\n$str1,\t年龄：$age\t薪资：${sal * 1.2}\t身高：$height")

  }
}

核心：打包

注意：顺序必须要这样，clean(maven) -> 编译（那个锤子敲一下） -> package(maven)

第一步，clear 保证清空

第二步，编译，这个时候会把java and scala 编译完成

第三步，确认了，编译出 scala and java 的 class 才打包才会成功，注意包内要有 lib 包含 scala的依赖
